# IF一覧

## 1. 文書の目的
【template-guidance】<br>
- 必須: 内部IF設計の入口文書であることを書く
- 任意: 一覧化する境界の種類を補足する
- 書かない: 各IFの詳細契約本文

【/template-guidance】<br>

本書は、〇〇システムで定義する内部IFを一覧化し、個別IF設計への入口を提供することを目的とする。

## 2. 前提
【template-guidance】<br>
- 必須: 対象が内部境界であり、外部向けIFではないことを書く
- 任意: 同期 / 非同期の扱いを補足する
- 書かない: 外部IF仕様の再掲

【/template-guidance】<br>

- 対象は、同一システム内のモジュール間 API、イベント、キュー、ジョブ境界、RPC などの内部IFとする。

## 3. 収集対象
【template-guidance】<br>
- 必須: 何を 1 IF とみなすかを明記する
- 任意: 詳細ファイルへ展開する基準を補足する
- 書かない: 境界のない単純な関数呼出

| 1 IF の単位 | 〇〇 |
