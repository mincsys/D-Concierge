# Install List Checklist

## 最初の提示に含める項目

- 項目名
- 用途
- 根拠文書
- `sudo` 要否
- ユーザ実行が必要かどうか

## 含める対象

- 実行に必要なランタイム
- 依存管理やパッケージ管理ツール
- ビルドや配備に必要なツール
- テスト、lint、format、type check の実行ツール
- DB やコンテナ実行基盤
- 外部サービスを使うための前提設定

## 含めない対象

- docs に根拠がない任意ツール
- 利便性だけの個人設定
- 将来使うかもしれない未確定ツール

## 提示時の注意

- 許可依頼より前に必ず提示する。
- `sudo` が必要な項目は、スキル側で実行せずユーザ依頼に回す。
- 未確定事項はインストール済み前提として扱わない。
