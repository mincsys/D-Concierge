# テンプレート索引

## 1. 使い方

- この索引を起点に、要件定義書とユーザ指定に合わせて必要なテンプレートだけを選ぶ。
- `必須` は原則採用する。
- `任意` は該当条件があるときだけ採用する。
- `汎用ひな型` は具体名へ複製して使い、そのまま成果物へ残さない。

| --- | --- | --- | --- |
| `01_システム構成/システム概要.md` | 必須 | 全体像、利用者、構成要素、外部連携を示すとき | 01_システム構成一式 |

| `08_共通設計/用語集.md` | 任意 | 独自用語、共通定義、略語を整理したいとき | 全文書 |

## 3. 採用時の注意

- `【template-guidance】` ブロックは作業ガイドであり、最終成果物では削除する。
- `〇〇` や `外部サービスA` などのプレースホルダは必ず置換する。
- 汎用ひな型は具体名へ複製したうえで編集する。
