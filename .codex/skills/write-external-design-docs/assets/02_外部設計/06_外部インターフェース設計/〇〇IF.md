# 〇〇IF

## 1. 文書の目的
【template-guidance】<br>
必須: 当該 IF 相手との連携を定義する目的、対象とする連携群、文書の位置付けを書く。<br>
任意: 関連業務や利用者影響を補足してよい。<br>
書かない: システム内部実装や製品選定理由。<br>
【/template-guidance】<br>

本書は、〇〇システムと〇〇の間で利用するインターフェース群の仕様を定義することを目的とする。

## 2. 前提
【template-guidance】<br>
必須: 接続前提、運用前提、識別子や時刻の扱いなど共通前提を書く。<br>
任意: 認証、暗号化、再送、監視などの基本方針を補足してよい。<br>
書かない: 秘密情報の実値。<br>
【/template-guidance】<br>

- 当該相手との連携は定められた方式で実施する。
- 識別子、時刻、コード値の扱いはシステム共通ルールに従う。
