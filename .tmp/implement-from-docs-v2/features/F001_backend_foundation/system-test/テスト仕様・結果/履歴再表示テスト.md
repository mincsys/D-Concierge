# 履歴再表示テスト

## 1. 文書の目的

本書は、総合テスト方針で定義したBIZ-03 履歴再表示に対応するテストケース、実施結果および証跡を記録することを目的とする。

## 2. 前提

- 本書は、`総合テスト方針.md` に基づいて作成する。
- 本書の対象は、チャット履歴一覧から過去チャットを選択し、保存済みのユーザ指示、回答、状態、参照元、Codex成果物を再表示する業務とする。
- 履歴表示では、保存済み回答と保存済みCodex成果物を使用する。
- 継続中のチャット実行処理を履歴から開いた場合は、保存済み状態を復元し、対象SSEへ再接続する。
- 同じテスト対象を `チャット実行テスト.md`、`キャンセルテスト.md` へ重複記載しない。

## 3. テスト対象

| 対象区分 | 対象 | 確認目的 |
| --- | --- | --- |
| 業務シナリオ | BIZ-03 履歴再表示 | 利用者が履歴一覧から過去チャットを選択し、保存済み内容を再表示できることを確認する |
| 画面 | チャット画面、参照元ビューア | 履歴一覧、履歴詳細、参照元表示、サイドバー操作、継続指示を確認する |
| API/SSE | IF-SB-04、IF-SB-05、IF-SB-06、IF-SB-08、IF-SB-09 | 履歴一覧取得、履歴詳細取得、継続中runのSSE再接続、保存済み成果物取得、参照元取得を確認する |
| データ | PostgreSQL、共有データソース、Codex成果物保存領域 | 保存済みチャット、回答、参照元、Codex成果物、終端状態が再表示できることを確認する |
