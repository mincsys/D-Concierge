# 認証テスト

## 1. 文書の目的

本書は、総合テスト方針で定義したBIZ-05 認証に対応するテストケース、実施結果および証跡を記録することを目的とする。

## 2. 前提

- 本書は、`総合テスト方針.md` に基づいて作成する。
- 本書の対象は、未ログイン時の保護、アカウント登録、ログイン、ログイン状態保持、ログアウト、セッション期限切れを確認する業務とする。
- Cookie、ログインセッション、有効期限、HTTP応答の確認は、Playwrightによる画面確認に加えて、HTTP確認とDB確認で行う。
- 同じテスト対象を `アカウント管理テスト.md` へ重複記載しない。

## 3. テスト対象

| 対象区分 | 対象 | 確認目的 |
| --- | --- | --- |
| 業務シナリオ | BIZ-05 認証 | 利用者がアカウント登録、ログイン、ログアウトを行い、ログイン状態を保持または解除できることを確認する |
| 画面 | ログイン画面、アカウント登録画面、開始画面、設定ダイアログ | 未ログイン表示、登録導線、ログイン導線、ログアウト導線、ログイン後遷移を確認する |
| API | 認証状態確認、アカウント登録、ログイン、ログアウト | API応答、Cookie、ログインセッション、利用者向けエラー表示を確認する |
| データ | ユーザ、ログインセッション | 登録、ログイン、ログアウト、期限切れセッション削除、複数セッションの扱いを確認する |
