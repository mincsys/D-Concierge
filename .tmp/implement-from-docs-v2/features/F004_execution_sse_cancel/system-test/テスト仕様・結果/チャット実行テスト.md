# チャット実行テスト

## 1. 文書の目的

本書は、総合テスト方針で定義したBIZ-01 チャット実行処理に対応するテストケース、実施結果および証跡を記録することを目的とする。

## 2. 前提

- 本書は、`総合テスト方針.md` に基づいて作成する。
- 本書の対象は、開始画面またはチャット画面でユーザ指示を送信し、検証済み回答、エラー、キャンセル済み、タイムアウトのいずれかに到達するチャット実行処理とする。
- Codex実行コンテナを使用する正常系では、回答内容と参照元の業務妥当性を手動で確認する。
- 画面から観測できる異常系は、ChromeのPlaywrightで操作と表示を確認する。
- 同じテスト対象を `キャンセルテスト.md`、`履歴再表示テスト.md` へ重複記載しない。

## 3. テスト対象

| 対象区分 | 対象 | 確認目的 |
| --- | --- | --- |
| 業務シナリオ | BIZ-01 チャット実行処理 | ユーザ指示送信から回答表示または利用者向け終端表示までの業務完了を確認する |
| 画面 | 開始画面、チャット画面、参照元ビューア | 入力、送信、状態表示、回答表示、参照元表示、成果物表示、エラー表示を確認する |
| API/SSE | IF-SB-01、IF-SB-02、IF-SB-03、IF-SB-06、IF-SB-08、IF-SB-09 | 画面操作を起点としたREST受付、SSE配信、参照元取得、成果物取得を確認する |
| 外部実行 | Codex実行コンテナ | 生成用Codex実行コンテナと検証用Codex実行コンテナを含む実行完了を確認する |
