# D-Concierge

D-Concierge は、登録済みデータソースを参照しながら、ユーザ指示に対して調査・分析・回答を行うチャット型アプリケーションです。

https://github.com/user-attachments/assets/83e6c73b-c8cb-4505-a170-00fb9897073e

## 主な機能

- チャット形式でのユーザ指示受付
- 登録済みデータソースを参照した回答生成
- 回答の検証と参照元表示
- チャット履歴の再表示
- アカウント登録、ログイン、設定変更
- チャット削除、アカウント削除

詳細は [機能一覧](docs/02_外部設計/03_機能設計/機能一覧.md) を参照してください。
