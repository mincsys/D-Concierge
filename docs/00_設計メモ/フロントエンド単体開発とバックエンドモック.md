# フロントエンド単体開発とバックエンドモック

## 目的

本メモは、Playwright CLIによるフロントエンド単体確認時の画面確認方法を整理する。

## Playwright確認

Playwright CLIでフロントエンド単体の画面確認を行う場合は、開発環境構築書のモック付き開発サーバ手順で対象サーバを起動し、Chromeで開く。

実バックエンドとの結合状態を確認する場合は、開発環境構築書の実バックエンド接続手順で対象サーバを起動し、同じ画面操作を確認する。

個別の異常系や境界値だけを差し替えたい場合は、Playwright routeで対象確認内だけレスポンスを上書きする。
