# accountApiモジュール設計

## 1. 文書の目的

本書は、`accountApi` モジュールの責務、不変条件、公開関数、およびエラー変換を定義することを目的とする。

## 2. 前提

- 本書の対象は `モジュール一覧.md` で詳細設計対象とした `accountApi` モジュールのみとする。
- 通信はブラウザ標準の `fetch` を直接使用し、共通HTTPクライアントは導入しない。
- API項目名は外部IFのsnake_case、画面モデルはTypeScript側のcamelCaseで扱う。

## 3. 責務

- 認証状態確認、ログイン、アカウント登録、ログアウト、ユーザ名変更、パスワード変更、アカウント削除APIを呼び出す。
- API応答のユーザ情報を画面モデル `AccountUser` へ変換する。
- APIエラー応答を `AccountApiError` へ変換する。
- APIの `field_errors` を画面フォームの項目キーへ変換する。
- 認証エラーかどうかを判定できる補助関数を提供する。

## 4. 不変条件

- `accountApi` は `backend_mock` を直接参照しない。
- フォーム入力制約の最終判定はAPI応答を正とし、画面側では文字数など即時に判断できる制約だけを補助的に扱う。
- HTTP 2xx以外は必ず `AccountApiError` として呼出元へ返す。
- CookieはブラウザとバックエンドのHTTP境界に委ね、画面モデルへログインセッショントークンを保持しない。

## 5. 公開プロパティ

| 名前 | 型 | 役割 | 読み取り専用 |
| --- | --- | --- | --- |
| `AccountApiError` | Error派生クラス | HTTPステータス、APIエラーコード、項目別エラー、利用者向けメッセージを保持する | はい |

## 6. 公開メソッド

| メソッド | 役割 | 入力 | 出力 | 事前条件 | 事後条件 |
| --- | --- | --- | --- | --- | --- |
| `getCurrentUser` | 認証状態を確認する | なし | 現在ユーザ | Cookie送信可能な同一オリジンAPIであること | ログイン中なら `AccountUser` を返し、未ログインなら `AccountApiError` を送出すること |
| `login` | ログインする | ユーザID、パスワード | 現在ユーザ | 入力値がフォームから取得済みであること | ログイン成功時に `AccountUser` を返すこと |
| `registerAccount` | アカウント登録する | ユーザID、ユーザ名、パスワード、パスワード確認 | 現在ユーザ | 入力値がフォームから取得済みであること | 登録成功時に `AccountUser` を返すこと |
| `logout` | ログアウトする | なし | なし | ログアウトAPIを呼び出せること | 成功時に呼出元がログイン画面へ遷移できること |
| `changeUserName` | ユーザ名を変更する | 新しいユーザ名 | 更新後ユーザ | 認証済みであること | 更新後の `AccountUser` を返すこと |
| `changePassword` | パスワードを変更する | 現在のパスワード、新しいパスワード、新しいパスワード確認 | なし | 認証済みであること | 成功時に呼出元が設定一覧へ戻れること |
| `deleteAccount` | アカウント削除を受け付ける | なし | 削除受付状態 | 認証済みであること | 成功時に呼出元がログイン画面へ遷移できること |
| `isAccountApiError` | アカウントAPIエラーか判定する | 任意のエラー | 判定結果 | なし | `AccountApiError` だけをtrueにすること |
| `isUnauthorizedAccountError` | 未ログイン/セッション切れか判定する | 任意のエラー | 判定結果 | なし | HTTP 401 の `AccountApiError` だけをtrueにすること |

## 7. 公開イベント

| イベント名 | 発火条件 | 通知内容 |
| --- | --- | --- |
| API成功 | 各APIが2xx応答を返したとき | 呼出元へ画面モデルまたは成功完了を返す |
| API失敗 | 各APIが2xx以外を返したとき | `AccountApiError` を送出し、呼出元が項目別エラーまたは共通メッセージを表示できるようにする |
