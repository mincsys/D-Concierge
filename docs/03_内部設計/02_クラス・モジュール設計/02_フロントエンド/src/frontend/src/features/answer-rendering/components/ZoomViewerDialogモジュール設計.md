# ZoomViewerDialogモジュール設計

## 1. 文書の目的

本書は、`ZoomViewerDialog` モジュールの責務、不変条件、公開プロパティ、公開関数、および公開イベントを定義することを目的とする。

## 2. 前提

- 本書の対象は `モジュール一覧.md` で詳細設計対象とした `ZoomViewerDialog` モジュールのみとする。
- 表示対象コンテンツのDOM生成と固有サイズ計算は、利用側モジュールが担当する。

## 3. 責務

- 全画面に近いダイアログ枠、ヘッダー、説明、閉じる操作を表示する。
- 拡大、縮小、全体表示の操作ボタンを表示する。
- マウスホイールによるズームとドラッグによるパン操作を提供する。
- 利用側から渡されたfit処理を、初期表示時、表示対象変更時、全体表示操作時に呼び出す。
- 表示領域と指定サイズに基づく全体表示用のtransform設定処理を、fit処理へ渡す。

## 4. 不変条件

- `open` の状態は親から受け取り、変更は `onOpenChange` で親へ通知する。
- 表示対象コンテンツのサイズ取得に失敗した場合でも、ダイアログ全体の表示を失敗させない。
- ズーム操作は定義済みの最小/最大倍率の範囲内で行う。
- 表示領域はダイアログの残り領域を占有し、ヘッダーと重ならない。

## 5. 公開プロパティ

| 名前 | 型 | 役割 | 読み取り専用 |
| --- | --- | --- | --- |
| `open` | 真偽値 | ダイアログ開閉状態 | はい |
| `title` | 文字列 | ダイアログタイトル | はい |
| `description` | 文字列 | スクリーンリーダー向け説明 | はい |
| `controlsLabel` | 文字列 | 表示操作領域のラベル | はい |
| `zoomInLabel` | 文字列 | 拡大ボタンのラベル | はい |
| `zoomOutLabel` | 文字列 | 縮小ボタンのラベル | はい |
| `fitLabel` | 文字列 | 全体表示ボタンのラベル | はい |
| `fitTooltip` | 文字列 | 全体表示ボタンのツールチップ | はい |
| `fitKey` | 文字列 | 初期fitを再実行する表示対象キー | はい |
| `onFitToView` | 関数 | 表示対象固有の全体表示処理 | はい |
| `onOpenChange` | 関数 | ダイアログ開閉変更の通知先 | はい |
| `children` | React要素 | ズーム対象コンテンツ | はい |

## 6. 公開メソッド

| メソッド | 役割 | 入力 | 出力 | 事前条件 | 事後条件 |
| --- | --- | --- | --- | --- | --- |
| `ZoomViewerDialog` | 共通ズームビューアを描画する | 開閉状態、タイトル、操作ラベル、fit処理、ズーム対象コンテンツ | React要素 | `onOpenChange` と `onFitToView` が渡されていること | `open` がtrueの場合、ズーム操作可能なダイアログが表示されること |

## 7. 公開イベント

| イベント名 | 発火条件 | 通知内容 |
| --- | --- | --- |
| `onOpenChange` | ダイアログを閉じる、または開閉状態が変わったとき | 変更後の開閉状態 |
| `zoomIn` | 拡大ボタンを選択したとき | 表示倍率を上げる |
| `zoomOut` | 縮小ボタンを選択したとき | 表示倍率を下げる |
| `fitToView` | 全体表示ボタンを選択したとき | 利用側のfit処理を呼び出す |
