# pageRangeモジュール設計

## 1. 文書の目的

本書は、`pageRange` モジュールの責務、不変条件、公開プロパティ、公開関数、および公開イベントを定義することを目的とする。

## 2. 前提

- 本書の対象は `モジュール一覧.md` で詳細設計対象とした `pageRange` モジュールのみとする。
- 入力のページ番号妥当性はバックエンドおよびPDFビューア側で検証し、本モジュールは表示文字列生成に限定する。

## 3. 責務

- PDF参照元の開始ページと終了ページから表示ラベルを生成する。
- 単一ページと範囲ページで表示形式を切り替える。

## 4. 不変条件

- 開始ページと終了ページが同じ場合は `p.開始ページ` 形式で返す。
- 開始ページと終了ページが異なる場合は `p.開始ページ-終了ページ` 形式で返す。
- 入力オブジェクトの値は変更しない。

## 5. 公開メソッド

| メソッド | 役割 | 入力 | 出力 | 事前条件 | 事後条件 |
| --- | --- | --- | --- | --- | --- |
| `formatPdfPageRange` | PDFページ範囲ラベルを生成する | locatorを持つ参照元 | ページ範囲ラベル | `locator.page_start` と `locator.page_end` が数値であること | 単一ページまたは範囲ページのラベルが返ること |
