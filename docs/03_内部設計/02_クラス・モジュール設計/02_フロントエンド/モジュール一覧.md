# モジュール一覧

## 1. 文書の目的

本書は、D-Conciergeのフロントエンドで設計対象とする主要モジュールを一覧化し、詳細設計を作成するモジュールと詳細設計を省略するモジュールを整理することを目的とする。

## 2. 前提

- 対象は `src/frontend/src` 配下のアプリケーション固有モジュールとする。
- React、Radix UI、shadcn/ui、Vite、Tailwind CSS、pdf.js、Mermaidなどのライブラリ提供機能そのものは対象外とする。
- `src/frontend/src/components/ui/` 配下はアプリケーション内で管理するが、shadcn/ui由来の薄いラッパーであるため詳細設計を省略する。
- `src/frontend/src/styles/globals.css` はテーマ、body初期設定、外部ライブラリ生成DOM向けCSSであり、共通設計で扱う。
- `src/frontend/src/assets/` 配下の画像ファイルは静的アセットであり、モジュール一覧の詳細対象から除外する。
- 詳細設計を書くモジュールは、モジュール名を対応する詳細文書へのリンクにする。
