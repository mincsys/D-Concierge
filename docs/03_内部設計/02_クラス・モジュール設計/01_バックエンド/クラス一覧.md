# クラス一覧

## 1. 文書の目的

本書は、D-Conciergeのバックエンドで設計対象とする主要クラスを一覧化し、詳細設計を作成するクラスと詳細設計を省略するクラスを整理することを目的とする。

## 2. 前提

- 対象は、アプリケーション固有に定義するバックエンドクラスのみとする。
- FastAPI、SQLAlchemy、Pydantic、Alembicなどのフレームワーク標準クラスとライブラリ提供クラスは対象外とする。
- Pydanticスキーマ、SQLAlchemy ORMモデル、単純な値オブジェクト、Protocolだけの抽象は一覧へ記載するが、詳細ファイルを作らず除外理由を示す。
- 詳細設計を書くクラスは、クラス名を対応する詳細文書へのリンクにする。
