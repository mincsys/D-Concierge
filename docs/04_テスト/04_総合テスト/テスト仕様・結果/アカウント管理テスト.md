# アカウント管理テスト

## 1. 文書の目的

本書は、総合テスト方針で定義したBIZ-06 アカウント管理に対応するテストケース、実施結果および証跡を記録することを目的とする。

## 2. 前提

- 本書は、`総合テスト方針.md` に基づいて作成する。
- 本書の対象は、設定ダイアログからユーザ名変更、パスワード変更、ログアウト確認、アカウント削除を行う業務とする。
- アカウント削除の確認では、ユーザ、ログインセッション、チャット関連DBデータ、生成用作業ディレクトリ、検証用作業ディレクトリ、保存済みCodex成果物実体、共有データソース、トレースログの状態を確認する。
- 同じテスト対象を `認証テスト.md`、`チャット削除テスト.md` へ重複記載しない。
